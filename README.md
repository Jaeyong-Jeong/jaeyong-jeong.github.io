# Jaeyong Jeong CV Website

Static CV website for GitHub Pages.

## Publish on GitHub Pages

1. Create a GitHub repository and push these files.
2. In the repository settings, open **Pages**.
3. Select the `main` branch and the root folder.
4. Save. GitHub will publish the site after the Pages build completes.

The public page entry point is `index.html`, and the downloadable CV is `CV_Jeong.pdf`.
